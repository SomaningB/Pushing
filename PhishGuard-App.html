📖 Project Overview

About PhishGuard

A hackathon project tackling phishing awareness through real-time detection and interactive education — built for students, by students.

🎯

The Problem

Over 3.4 billion phishing emails are sent daily. Students are primary targets — 1 in 4 click malicious links without knowing it.

⚡

Our Solution

Real-time URL and message analysis using heuristic rules + ML scoring. Instant risk score with human-readable explanations.

🧠

Detection Engine

18+ detection signals: HTTPS check, domain age, homograph attacks, suspicious TLDs, keyword density, IP-based URLs, and more.

🎮

Education Layer

Gamified quiz challenges teach users to spot phishing patterns — making security awareness engaging and memorable.

🏗️ System Architecture

[ USER INPUT ] → URL / Email Message │ ▼ [ PREPROCESSING ] • URL Parsing (protocol, domain, path, params) • Text extraction & tokenization • Feature vector construction │ ▼ [ DETECTION ENGINE ] ┌──────────────────────────────────────┐ │ Rule Layer (18 heuristic checks) │ │ ├─ Protocol check (HTTP vs HTTPS) │ │ ├─ Domain analysis (TLD, length) │ │ ├─ Homograph & typosquat detection │ │ ├─ Suspicious keyword scanner │ │ ├─ IP-based URL detection │ │ └─ Special char & entropy analysis │ │ │ │ Score Aggregator (weighted sum) │ │ → Risk Score: 0–100 │ └──────────────────────────────────────┘ │ ▼ [ RISK CLASSIFIER ] 0–30 → 🟢 SAFE 31–60 → 🟡 SUSPICIOUS 61–100→ 🔴 DANGEROUS │ ▼ [ OUTPUT LAYER ] • Risk Score + Badge • Threat findings (itemized) • Recommendation • Educational tip
